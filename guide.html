Guide - How to configure Miyuchan for your discord server

This guide is here to help people in order for them to configure Miyuchan for their discord server, step-by-step.

I - Configuration panel

By doing /configuration you can access to the configuration panel. It's here where the magic happens.

You can edit everything you want there such as the Report Channel, Level channel, Welcome channel, Auto-nick, Auto-role, Economy system and the shop

II - Editing the channels

The first time you see the configuration panel, every options featuring channels will be set to #None

#None means that the option isn't enabled (except for levels, the default channel will be the current one)

By clicking on the orresponding buttons, a discord modal will show up asking you the name of a channel (without the # symbol)

Onnce you press 'Send', the option will be enabled with the channel you just set.

III - How to configure the shops or enable the economy system

Some commands require the Economy System to be enabled, such as /claim, /colors, /shop etc. To enable it, go to the configuration panel, and click on the Economy button.

Once enabled you're ready to go! Commands requiring it will now work perfectly. Now let's edit the shop.

You can edit the items available in the shop in the configuration panel by clicking on it button Edit shop. You will be asked if you want to add or remove an item, the item name/role, the cost

Of course if you've decided to remove an item, there is no need to specify a cost in points.

You're ready to go!
